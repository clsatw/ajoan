Iot question 101: How to access devices from anywhere?

It is easy to connect esp8266 devcies to the net; however, there are some problems need to be resolved:

  • When programming, we usually hard code ssid and password to the devie for wifi connection, but what if we take it to somewhere else or we change our router's ssid or password?
  • The device is assigned a dynamce private ip adress by DHCP servers and doesn't have a DNS name, something likes www.abc.com, after connection. How can we acess it from anywhere?
  • How can i trigger an event for the device to take an action. e.g, I want the device to turn on/off aquarium waterpump at a certain tim automatically.
  • Last but not least, we need to put the device into sleeping mode to save power.

Something to the rescue

After studying and pulling most my hair out, the follwoing methods are implemented to solve the issues:

  • 3rd party wifi manager library
  • MQTT (Message Queuing Telemetry Transport)
  • IFTTT (If This Then That)
  • interrupt and deep_sleep mode in arduion
