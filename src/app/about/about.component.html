Iot question 101: How to access devices from anywhere?

It is easy to connect esp8266 devcies to the net; however, there are some questions need to be resolved:

1) Usually we hard code ssid and password to the devie for wifi connection. What if we take it to somewhere else or we change our router's ssid or password? 2) The device will be assigned a dynamce private ip adress by DHCP and doesn't have a DNS name after connection. How can we acess it from the net? 3) How can i trigger an event for the device to take action. e.g, i want the device to turn on/off aquarium waterpump at a certain time. 4) Last but not least, we need to put the device into sleeping mode to save power.

Something to the rescue

After studying and pulling most my hair out, the follwoing methods are implemented to solve the issues: 1) wifi manager library 2) MQTT 3) IFTTT 4) interrupt and deep_sleep mode in arduion
